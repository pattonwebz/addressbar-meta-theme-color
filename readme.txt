Changes the address bar colour for users with mobile browsers (works in Chrome and Opera).

== Description ==

Use this plugin to output the required meta tag to change the address bar color in various mobile browsers.

== Installation ==

1. Upload `the plugin folder` to the `/wp-content/plugins/` directory or install from inside the admin interface.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the customizer to select the color you want the address bar to appear for mobile users.
